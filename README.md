# Radar Ultrasónico con ESP32 y MQTT

Proyecto realizado con ESP32 y MicroPython utilizando:

- Sensor HC-SR04
- Servo motor
- MQTT
- Adafruit IO
- Wokwi

## Funciones

- Encendido y apagado remoto del radar
- Detección de objetos
- Configuración de distancia mediante slider
- Publicación de eventos MQTT
- Movimiento automático del servo

## Tecnologías utilizadas

- MicroPython
- MQTT
- ESP32
- Adafruit IO
- Wokwi

## Archivos

- `main.py` → lógica principal
- `hcsr04.py` → driver del sensor ultrasónico
- `diagram.json` → conexiones del circuito
